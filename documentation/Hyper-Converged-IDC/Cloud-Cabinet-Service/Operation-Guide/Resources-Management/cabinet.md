# 机柜

1. 打开控制台，选择超融合数据中心->云托管服务->资源管理->机柜，进入机柜列表页，展示京东云为客户分配的机柜信息，如下图；
 ![机柜列表查看连接](https://github.com/jdcloudcom/cn/blob/cn-Cloud-Cabinet-Service/image/Hyper-Converged-IDC/Cloud-Cabinet-Service/CCS002.png)
2. 机房：默认选择某一个机房，如果有多个机房，支持不同机房类型切换选择，切换后列表显示不同机柜信息；
 ![地域区查看连接](https://github.com/jdcloudcom/cn/blob/cn-Cloud-Cabinet-Service/image/Hyper-Converged-IDC/Cloud-Cabinet-Service/CCS002.png)
