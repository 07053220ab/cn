# 计费规则
## 按套餐包计费
根据购买的套餐包计算费用，可购买月套餐包、季套餐包，半年套餐包和年套餐包。

## 按购买量打折
根据每次购买的量打不同的折扣。
* **中国联通**
500-1000张 2.2折

1001-5000张 2.1折

5001张起1.9折
* **中国移动**
500-1000张 2.3折 

1001-5000张 2.2折 

5001张起2.0折
* **中国电信**
500-1000张1.8折 

1001-5000张 1.7 折

 5001张起1.5折
## 示例
您在2019年5月1日购买了1G/月/4元的套餐包，您需要支付的费用是4元，该物联网卡有效期到2019年5月31日，过期后，您可以继续续订套餐，续订的价格和购买时的价格一致。
