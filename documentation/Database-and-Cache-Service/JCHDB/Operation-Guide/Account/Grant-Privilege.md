# 设置权限

## 权限说明
DRDS新增了两个账号权限，用于设置读写分离相关的操作，具体的账号权限说明如下：
- rw：具有该数据库主节点的读写权限，所有SQL发往RDS MySQL主节点。
- ro：具有该数据库主节点的读权限，所有SQL发往RDS MySQL主节点。

## 操作说明
#### 1. 进入【账号管理】页面

点击实例名，进入实例页面，选择 **【账号管理】** 页面，点击“**修改权限”**

![账号列表](../../../../../image/DRDS/account-list.png)


#### 2. 设置权限

设置账号的权限信息
