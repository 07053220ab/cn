# 产品规格

边缘计算产品是运行在本地终端的软件包，因此对终端硬件环境有如下要求：

|    名称    |        规格        |
| :--------: | :----------------: |
|   处理器   |       x86-64       |
|    主频    |      ≥1.6GHz       |
|    内存    |        ≥2GB        |
|  硬盘空间  |       ≥64GB        |
|  操作系统  | Ubuntu 18.04及以上 |
| Docker版本 |      ≥v18.06       |
