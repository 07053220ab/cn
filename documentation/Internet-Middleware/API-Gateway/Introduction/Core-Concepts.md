# 核心概念
以下是API网关帮助文档中使用到的概念及其解释，请参考。

| 概念 | 解释 |
| :- | :- |
| AK/SK | Access Key/Access Key Secret京东云用户的密钥.通常为API提供者使用，用于校验网关与API提供者服务器的通信。|
| APIKey/APISecret | API调用者的密钥用于API提供者使用API时，网关校验API调用者的访问时使用。 |
