# 自定义登录样式

开发者可以通过工单提交池Uid、应用ID、应用Logo、登录页底图等信息替换用户池应用的登录页元素。默认登录样式如下图：

![默认登录样式](../../../../image/IAS/Application-Management/5default-login.png)

手机端自适应页面样式：

![默认登录样式-手机端](../../../../image/IAS/Application-Management/6default-login.png)

提交的页面元素要求如下：
1. 应用Logo
2. 登录页底图
3. 登录按钮色值
4. 输入框内提示文案
5. 注册页面URL
提交后将于3-5个工作日进行处理。
