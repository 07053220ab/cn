# 用户池概述

用户池是一款账号托管产品，支持自定义用户信息、密码强度设置、认证安全策略、用户登录方式设置、关联应用、自定义登录/忘记密码页面样式、关联第三方登录服务提供商等功能，为应用开发者提供了完备的账号体系服务。

目前京东智联云支持通过OpenAPI创建和管理用户池及池内的用户，也开放HTTPS接口支持用户访问令牌直接调用，以实现用户中心功能。
