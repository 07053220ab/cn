# 添加印章

用户可以通过京东云控制台及OpenAPI来添加印章。

### 控制台添加印章

在印章管理页面中，点击“添加印章”按钮。

![1576584564982](C:\Users\liangzhiyong3\AppData\Roaming\Typora\typora-user-images\1576584564982.png)

在弹窗中完善密钥信息，包括印章名称，上传印章文件；填写完成后点击确定，完成印章的添加。
