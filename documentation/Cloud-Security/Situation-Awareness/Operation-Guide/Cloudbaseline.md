# 云产品基线

### 功能模块定义

从身份认证、网络访问控制、数据安全、日志审计、基础安全防护五个维度为您提供云产品安全配置的检测，帮助您及时发现您的云产品配置风险并提供相应的修复方案。

#### 云产品基线操作

本次云产品基线提供11款云产品安全配置核查，包括：云主机、原生容器、私有网络、负责均衡、云数据库RDS、云数据库MongoDB、IP高防、Web应用防火墙、主机安全、访问控制、操作审计。
