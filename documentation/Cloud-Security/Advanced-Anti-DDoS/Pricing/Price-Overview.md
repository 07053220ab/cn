# 计费说明

计费单位：人民币（RMB） 
计费项：保底防护＋弹性防护+业务带宽，其中弹性防护为可选项。
保底防护：按月/年计费，根据线路的不同，价格不同；
弹性防护：按天后付费；
业务带宽：按月/年计费，默认赠送100Mbps；
保底防护+业务带宽，按统一的包年/包月方式计费，只可升级，不可降级。
