标准存储：

<div id="user-content-1"></div>

标准存储（Standard）为用户提供了高可靠性、高可用性、高性能的对象存储服务,能够支持频繁的数据访问。
标准存储拥有低访问时延与高吞吐量，因而适用于有大量热点文件。标准存储类型是各种社交、分享类的图片、大型网站、音视频应用、大数据分析的合适选择。

关键特性：
* 9.9999999999%的数据可靠性
* 按照99.99%服务可用性设计
* 支持实时访问
* 支持HTTPS加密传输
* 支持图片处理
* 支持音视频处理
* 支持地域：全部地域
* 存储费用：标准
* 无最短存储时间和最小计量空间
