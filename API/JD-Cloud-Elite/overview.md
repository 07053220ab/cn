## API
|接口名称|请求方式|功能描述|
|---|---|---|
|**confirmSaleServiceDelivery**|POST|确认交付|
|**getSaleServiceByDeliverNumber**|GET|根据交付单号查询交付单信息|
|**getStoreService**|GET|获取云存服务信息|
|**jdxQueryPrice**|POST|查询价格|
|**jdxReportOrder**|POST|上报订单|
|**listSaleService**|GET|分页查询交付单信息|
